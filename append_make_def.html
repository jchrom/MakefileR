Appends a Makefile variable definition to a Makefile

Usage

append_make_def(makefile, variable, definition)

Arguments

makefile
A Makefile created by makefile
variable
Variable name
definition
Definition for this variable

Value

The first parameter, with the newly created rule appended

Description

This helper function creates a variable definition and appends it to an existing Makefile. Most useful in pipes.

Examples

library(magrittr)
makefile() %>%
  append_make_def("R_USER_LIBRARY", .libPaths()[[1L]])
